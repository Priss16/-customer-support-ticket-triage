# Customer Support Ticket Triage System

## Project Overview
This project is part of the **Week-1 Internship Assignment**.  
It implements a **Customer Support Ticket Triage System** using Python to automatically analyze customer support tickets.

The system:
- Cleans customer ticket descriptions
- Classifies issues into categories
- Assigns priority levels
- Calculates SLA response time
- Generates a summary report

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub

## Dataset
The dataset contains customer support ticket details.  
The **Ticket Description** column is used for text processing and analysis.
